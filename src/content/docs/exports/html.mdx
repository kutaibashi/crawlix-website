## Overview

The Interactive HTML export produces a single, self-contained HTML file that includes all your crawl data in a fully interactive web interface. No server, no app, no internet connection required to view it.

**Requires Pro or Agency tier.**

## Generating

Go to **File > Export > Interactive HTML**.

## What's Included

The HTML file contains:

- All crawled pages with full metadata
- All issues with filtering and sorting
- Charts and data visualizations (issue breakdown, severity distribution, CWV scores)
- Link analysis table
- Image analysis table
- Search functionality

## Interactivity

Viewers can:

- **Filter** by severity, category, or issue type
- **Sort** any column by clicking the header
- **Search** URLs or issue names
- **Toggle** between different data views

## Sharing

Send the single `.html` file to anyone — clients, colleagues, or stakeholders. They can open it in any web browser and interact with the full dataset without needing Crawlix installed.

This is one of the most convenient ways to deliver an SEO audit to a client who wants to explore the data themselves.

## File Size

Interactive HTML files are self-contained (all CSS, JS, and data embedded). A typical file is 2–15 MB depending on the number of pages and issues.
