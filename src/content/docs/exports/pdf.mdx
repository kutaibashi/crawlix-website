## Overview

Generate a professional PDF report suitable for client delivery. The PDF includes an executive summary, issue breakdowns, Core Web Vitals, and more — all in a clean, printable format.

**Requires Pro or Agency tier.**

## Generating a PDF Report

Go to **File > Export > PDF Report**.

## What's Included

- **Executive Summary** — headline metrics: total pages crawled, issue counts by severity, crawl date
- **Top Issues** — highest-priority errors and warnings with affected page counts
- **Issue Breakdown by Category** — sections for Technical SEO, On-Page, Content, Links, Images, etc.
- **Core Web Vitals** — LCP, CLS, FCP summary (if CWV was measured during the crawl)
- **Custom Sections** — any custom extraction data flagged with validation issues

## Customization Options

Before generating, you can configure:

- **Which sections to include** — exclude sections not relevant to this client
- **Site name** — appears in the report header
- **Report date** — defaults to crawl date
- **Introduction text** — add a brief overview or context paragraph

## Branding

By default, PDF reports include the Crawlix logo and "Powered by Crawlix" in the footer.

**Agency tier** users can replace this with their own branding — company name, logo, and colors. See [White-Label Branding](/docs/white-label/setup) for setup instructions.

## File Size

A typical PDF report for a 1,000-page crawl is 1–5 MB. Reports for very large sites with many issues may be larger.
