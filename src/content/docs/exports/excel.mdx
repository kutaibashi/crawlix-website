## Overview

The Excel export produces a formatted `.xlsx` file with multiple worksheets. Unlike a raw CSV, the Excel export includes formatting, color-coding, and column filters — ready to share with clients or team members.

**Requires Pro or Agency tier.**

## Exporting

Go to **File > Export > Excel (.xlsx)**.

## Worksheets Included

| Sheet | Contents |
|-------|----------|
| Summary | Total pages crawled, issue counts by severity and category, crawl date and settings |
| Pages | All crawled pages with full metadata |
| Issues by Category | Issues grouped and subtotaled by category |
| Links | All discovered links with status |
| Images | All images with alt text status and file info |
| Resources | Resource breakdown by page |

## Formatting

The Excel file includes:

- **Bold headers** in the first row of each sheet
- **Color-coded severity**: Error rows in red/orange, Warnings in yellow, Notices in blue
- **Column filters** on all data sheets (click the filter icon to sort or filter)
- **Frozen header rows** so headers stay visible while scrolling
- Auto-sized column widths for readability

## Sharing with Clients

The formatted Excel export is designed for sharing with clients or non-technical team members who prefer working in spreadsheets. It presents the data professionally without requiring Crawlix to be installed.

For an even more polished client deliverable, consider the [PDF Report](/docs/exports/pdf) or [Interactive HTML](/docs/exports/html) exports.
