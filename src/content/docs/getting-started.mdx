# Getting Started with Crawlix

Welcome to Crawlix, the privacy-first SEO crawler. This guide will help you get up and running in minutes.

## Installation

### Windows

1. Download the `.exe` installer from the [download page](/download)
2. Run the installer and follow the prompts
3. Launch Crawlix from your Start menu

### macOS

1. Download the `.dmg` file for your processor (Apple Silicon or Intel)
2. Open the DMG and drag Crawlix to Applications
3. Launch Crawlix from Applications

### Linux

1. Download the `.AppImage`, `.deb`, or `.rpm` package
2. For AppImage: Make it executable with `chmod +x` and run
3. For deb/rpm: Install using your package manager

## Your First Crawl

1. Enter a URL in the address bar (e.g., `https://example.com`)
2. Click **Start Crawl** or press Enter
3. Wait for the crawl to complete
4. Review the results in the dashboard

## Understanding Results

Crawlix organizes issues by severity:

- **Errors** (red): Critical issues that need immediate attention
- **Warnings** (yellow): Issues that should be addressed
- **Passed** (green): Checks that passed successfully
