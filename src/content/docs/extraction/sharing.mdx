## Overview

Agency tier users can publish extraction rule sets to share with the Crawlix community. Other users can browse and import shared templates.

**Requires Agency tier.**

## Publishing a Rule Set

1. Configure your extraction rules.
2. Select the rules you want to share.
3. Click **Publish to Community**.
4. Add a name, description, and tags (e.g., "Shopify", "WordPress", "Real Estate").
5. Click **Publish**.

Published rule sets are visible to all Crawlix users.

## Browsing Community Templates

All tiers can import community templates:

1. Go to **Settings > Custom Extraction > Browse Community Templates**.
2. Search by name, tag, or site type.
3. Click **Import** on any template.
4. The rules are added to your extraction configuration — review and adjust selectors as needed.

## Use Cases for Sharing

Agencies benefit from sharing in two ways:

- **Give back to the community** — share specialized extraction rules you've built for common CMS types
- **Onboard faster** — import community-built templates as a starting point for new client site types

## Editing After Import

Imported community templates are fully editable. The community template is a copy — your changes don't affect the original published template.

## Managing Your Published Templates

View and manage your published templates in **Settings > Custom Extraction > My Published Templates**. You can:

- Edit the name, description, or tags
- Update the rule set (publishes a new version)
- Unpublish to remove it from the community listing
