## Overview

**Quick Audit** is a Chrome extension that gives you instant SEO analysis of any webpage you're viewing — without running a full crawl.

## Installation

1. Open the **Chrome Web Store** in Google Chrome or any Chromium-based browser.
2. Search for **"Quick Audit Crawlix"**.
3. Click **Add to Chrome**.
4. Confirm the permissions requested:
   - `activeTab` — access the current tab's page content for analysis
   - `storage` — save your settings and preferences
5. The extension installs instantly.

## Pinning the Extension

For easy access:

1. Click the **puzzle piece** icon in the Chrome toolbar.
2. Find **Quick Audit** in the list.
3. Click the **pin** icon to keep it visible in your toolbar.

## Permissions

Quick Audit only accesses the tab you actively click the extension button on. It does not read other tabs or run in the background.

## Browser Compatibility

Quick Audit works in:

- Google Chrome
- Microsoft Edge (Chromium)
- Brave
- Other Chromium-based browsers

Firefox support is not currently available.

## No Account Required

Quick Audit works standalone for all basic analysis features. No Crawlix account or license is required to use the extension's core functionality.

## Pro/Agency Features

Some advanced features in the extension (like syncing pages to the Crawlix desktop app) require the Crawlix desktop app to be running on the same machine. See [Syncing with Desktop](/docs/browser-extension/sync).
