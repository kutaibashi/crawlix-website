## Command Palette

The fastest way to navigate Crawlix is the **command palette**.

Press **Cmd+K** (macOS) or **Ctrl+K** (Windows / Linux) to open it.

From the command palette, you can:

- Switch to any view (Pages, Issues, Links, Images, etc.)
- Open Settings
- Start or stop a crawl
- Search for any action or setting by name

The command palette supports fuzzy search — type a few letters of what you're looking for.

## Full Shortcut Reference

Crawlix has a built-in shortcut reference available at **Help > Keyboard Shortcuts** in the menu bar. This lists all current keyboard shortcuts for the installed version, so you always have an up-to-date reference.

## Tips

- Use the command palette to avoid clicking through menus for common tasks.
- Keyboard shortcuts may vary slightly between major versions — always check Help > Keyboard Shortcuts for the current list.
