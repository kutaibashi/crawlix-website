## Main Navigation Views

The left sidebar gives you access to all major views:

- **Pages** — every crawled URL with full metadata: status code, title, description, word count, and more. Search, sort, and filter by any column.
- **Issues** — all SEO problems found, grouped by severity (Error, Warning, Notice) and category (Technical, On-Page, Content, Links, Images, etc.). Filter and sort to prioritize fixes.
- **Links** — all discovered links (internal and external), their status codes, anchor text, and link types.
- **Images** — all images found during the crawl with alt text status, file size, and dimensions.
- **Core Web Vitals** — LCP, CLS, INP, FCP, and TTFB per page (requires JavaScript rendering enabled).
- **Structured Data** — JSON-LD blocks found on each page, validated against Schema.org types.
- **Resources** — breakdown of CSS, JavaScript, images, fonts, and other resources per page.
- **Site Structure** — an interactive graph visualizing your internal link structure.

## Issues View

The Issues view is often the best starting point after a crawl. You can:

- Filter by **severity**: Errors (highest priority), Warnings, Notices
- Filter by **category**: Technical SEO, On-Page, Content, Links, Images, Structured Data, Security, Accessibility, International
- Filter by **issue type** to find specific checks
- Click any issue to see which pages are affected
- Click any page to open its full detail view

## Pages View

The Pages view shows every crawled page as a row. Columns include URL, status code, title, meta description, H1, word count, links in, links out, and more. Use the column picker to show/hide columns relevant to your audit.

## Crawl Sessions

Each time you crawl a site, Crawlix creates a new session. Old sessions are preserved so you can compare results over time. Switch between sessions using the session picker at the top of the app.

## Command Palette

Press **Cmd/Ctrl+K** to open the command palette. You can search for views, settings, or actions by name — a fast way to navigate without using the mouse.
