## Windows

1. Go to [crawlix.app/download](/download) and download the `.exe` installer.
2. Run the installer and follow the on-screen prompts.
3. Launch Crawlix from the Start menu or desktop shortcut.
4. On first launch, accept the license agreement.
5. If you have a license key, go to **Settings > License > Enter License Key** to activate. Free tier users can skip this step.

## Linux

### AppImage (recommended — works on most distros)

1. Download the `.AppImage` file from [crawlix.app/download](/download).
2. Make it executable:

```
chmod +x Crawlix-*.AppImage
```

3. Run it directly — no installation required:

```
./Crawlix-*.AppImage
```

### DEB (Debian / Ubuntu)

```
sudo dpkg -i crawlix_*.deb
```

### RPM (Fedora / RHEL / openSUSE)

```
sudo rpm -i crawlix_*.rpm
```

## First Launch

When you open Crawlix for the first time:

- Accept the license agreement to proceed.
- You'll see the main window with an empty URL field.
- **Free tier** users can start crawling immediately — no account or key required.
- **Pro or Agency** users should activate their license key (Settings > License) to unlock full features.

## macOS

macOS support is coming soon. It is not yet available.

## No Account Required

Crawlix does not require you to create an account. The Free tier is fully functional without registration. License activation (for Pro/Agency) uses a license key sent to your purchase email.
