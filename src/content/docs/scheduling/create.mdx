## Overview

Scheduled crawls automatically run Crawlix on a site at a defined interval — daily, weekly, or monthly. This is ideal for ongoing SEO monitoring of client sites or your own properties.

**Requires Pro or Agency tier. The Crawlix app must be running at the scheduled time.**

## Creating a Schedule

1. Go to **Settings > Schedules > New Schedule**.
2. Choose a **site** from the list of previously crawled sites. (You must have crawled the site at least once before scheduling.)
3. Set the **frequency**: Daily, Weekly, or Monthly.
4. Set the **time**: hour and minute for when the crawl should start.
5. For weekly schedules, choose the **day of the week**.
6. For monthly schedules, choose the **day of the month**.
7. Click **Save**.

## First Run

After creating a schedule, you can choose:

- **Run immediately** — start the first crawl now
- **Wait for scheduled time** — start at the next occurrence of the scheduled time

## App Must Be Running

Scheduled crawls require Crawlix to be open and running at the scheduled time. If Crawlix is closed when a scheduled crawl is due, the crawl is skipped and will run at the next scheduled time.

Consider keeping Crawlix running in the background during expected crawl windows.

## Managing Schedules

View, edit, and manage all schedules in **Settings > Schedules**. See [Managing Schedules](/docs/scheduling/manage) for details.
